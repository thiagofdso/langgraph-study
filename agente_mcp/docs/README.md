# agente_mcp Docs

Este diretório concentra a documentação incremental do agente multi-servidor refatorado. O objetivo é registrar guias operacionais, tutoriais para adição de novos servidores MCP e roteiros de QA manual alinhados ao quickstart oficial.

## Estrutura inicial
- `README.md` (este arquivo): visão geral e links relevantes.
- `servers.md` (futuro): passo a passo para cadastrar/remover servidores declarativos.
- Outros artefatos podem ser adicionados conforme as fases de US1/US2 evoluem (por exemplo, logs de execução, troubleshooting avançado ou playbooks específicos).
